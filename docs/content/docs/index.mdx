# React Native AI Stack

React Native AI Stack is a comprehensive library that provides easy-to-use hooks and components for integrating MediaPipe's on-device AI capabilities into your mobile applications. Built with performance in mind, it leverages the power of MediaPipe Tasks and modern React Native patterns to bring high-performance machine learning features directly to your users' devices.

## Getting Started

To get started with React Native AI Stack, you'll typically:

1. **Install the library** in your React Native project
2. **Choose an AI task** you want to implement
3. **Import the relevant hook** (e.g., `useTextClassifier`, `useFaceDetector`)
4. **Provide a model source** (local asset, remote URL, or file path)
5. **Call the task-specific functions** (e.g., `classify`, `detect`, `embed`)

### Installation

```bash
npm install react-native-aistack
```

### Basic Usage Example

```typescript
import { useTextClassifier } from 'react-native-aistack';

const MyComponent = () => {
  const { classify, isLoading, error } = useTextClassifier(
    { bundle: 'model.tflite' },
    { maxResults: 5 }
  );

  const handleClassify = async () => {
    try {
      const result = await classify('Your text here');
      console.log('Classification result:', result);
    } catch (error) {
      console.error('Classification failed:', error);
    }
  };

  return (
    <View>
      <Button
        title="Classify Text"
        onPress={handleClassify}
        disabled={isLoading}
      />
    </View>
  );
};
```

## Model Sources

React Native AI Stack supports various ways to provide your AI models:

- **`UriModelSource`**: Load models from a remote URL with automatic downloading and caching
- **`BundleModelSource`**: Load models from your app's asset bundle (e.g., `android/src/main/assets`)
- **`FilePathModelSource`**: Load models from a local file path on the device

```typescript
// Remote URL with caching
const remoteModel = { uri: 'https://example.com/model.tflite' };

// App bundle asset
const bundleModel = { bundle: 'model.tflite' };

// Local file path
const fileModel = { filePath: '/path/to/model.tflite' };
```

## Performance Optimizations

React Native AI Stack is built with performance in mind:

- **Nitro Modules**: Uses Nitro for high-performance native module communication
- **ArrayBuffer**: Leverages `ArrayBuffer` for zero-copy data transfer
- **Asynchronous Operations**: All AI operations are asynchronous to prevent UI blocking
- **Model Caching**: Intelligent model caching to minimize load times
- **Memory Management**: Efficient memory usage with automatic cleanup

## Architecture

The library is built on top of:

- **MediaPipe**: Google's cross-platform framework for building multimodal applied ML pipelines
- **React Native**: For cross-platform mobile development
- **Nitro Modules**: For high-performance native module communication
- **TypeScript**: For type safety and better developer experience

## Best Practices

- Use `AssetSource` for large data transfers (images, audio)
- Implement proper error handling for AI operations
- Cache models locally for better performance
- Use async/await for all AI operations
- Monitor memory usage for long-running applications
